.horizontal(@start-color; @end-color; @start-percent; @end-percent;)

Horizontal gradient, from left to right.

Creates two color stops, start and end, by specifying a color and position for each color stop.

Color stops are not available in IE9 and below.

@start-color: #555; @end-color: #333; @start-percent: 0%; @end-percent: 100%

Example:

#gradient .horizontal(@start-color: #555; @end-color: #333; @start-percent: 0%; @end-percent: 100%)

.vertical(@start-color; @end-color; @start-percent; @end-percent;);

Vertical gradient, from top to bottom

Creates two color stops, start and end, by specifying a color and position for each color stop.

Color stops are not available in IE9 and below.

Example:

#gradient .vertical(@start-color: #555; @end-color: #333; @start-percent: 0%; @end-percent: 100%)

.directional(@start-color: #555; @end-color: #333; @deg: 45deg);

.horizontal-three-colors(@start-color: #00b3ee; @mid-color: #7a43b6; @color-stop: 50%; @end-color: #c3325f)

.vertical-three-colors(@start-color: #00b3ee; @mid-color: #7a43b6; @color-stop: 50%; @end-color: #c3325f)

.radial(@inner-color: #555; @outer-color: #333)

.striped(@color: #555; @angle: 45deg)
